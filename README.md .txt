# Application de Génération de Cartes de Contrôle

Cette application web utilise Flask pour transformer des données en cartes de contrôle avec des limites de contrôle et de surveillance.

## Fonctionnalités
- Upload de fichiers CSV contenant des données.
- Génération automatique de cartes de contrôle avec les limites UCL, LCL, et moyenne.
- Téléchargement des résultats sous forme de graphiques.
